# AI Lecturer Curriculum Generator

An AI-powered Next.js application that helps lecturers generate structured academic curricula using Gemini AI.

## Features
- Curriculum generation
- Lecturer-focused UI
- AI-powered workflow
- Production-ready deployment

## Tech Stack
- Next.js
- React
- Tailwind CSS
- Gemini AI
- Vercel

## Deployment
Deployed on Vercel.
